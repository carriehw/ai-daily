# AI・日報 (AI Daily Briefing)

每日 AI 精選快訊 — 涵蓋國際、香港、中國、台灣四大市場，重點追蹤 Claude / ChatGPT / Gemini 等模型與產品動態。

**線上閱讀：** https://carriehw.github.io/ai-daily/

## 內容分五大版塊
模型發布/更新 · 產品發布/更新 · 行業動態 · 論文研究 · 技巧與觀點

每條新聞附「點解重要」，以香港 AI 用家及投資者視角解讀影響。

## 更新方式
每朝自動抓取多個來源（smol.ai、各大官方 blog、國際及大中華媒體），
由 `data.json` 經 `build.py` 生成單檔 `index.html`。

```bash
PYTHONIOENCODING=utf-8 python build.py
```
